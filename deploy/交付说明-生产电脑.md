# 客服消息中台生产电脑交付说明

## 交付包内容

这个包是完整迁移包，包含：

- 中台接口服务：`apps/api`
- 中台网页：`apps/web`
- 邮箱 IMAP 采集器：`collectors/email-imap`
- 浏览器插件：`collectors/browser-extension`
- 本机邮件中转和中台邮箱入库预留模块：`collectors/local-mail-relay`、`collectors/hub-mail-ingest`
- 当前 SQLite 数据库：`apps/api/prisma/dev.db`
- 当前邮箱采集游标：`collectors/email-imap/.email-imap-state.json`
- 实际敏感配置：`apps/api/.env`、`collectors/email-imap/.env`
- 平台语音报警文件：`apps/web/public/alerts`
- 生产电脑启动脚本：`deploy/windows`

## 生产电脑安装

1. 把压缩包解压到生产电脑，例如：`D:\xiaoxijuhe`
2. 确认生产电脑已安装 Node.js 22 或更高版本。
3. 双击运行：`deploy\windows\install.bat`
4. 等待依赖安装、数据库同步和构建完成。

## 启动

双击运行：

`deploy\windows\start-all.bat`

启动后打开：

`http://localhost:5173/`

## 检查状态

双击运行：

`deploy\windows\status.bat`

它会检查：

- 网页是否打开
- 接口是否正常
- 网易 IMAP 状态接口是否正常
- API、网页、邮箱采集器进程是否存在
- 当前未查看新消息数量

## 停止

双击运行：

`deploy\windows\stop-all.bat`

## 浏览器插件安装

插件目录：

`collectors\browser-extension`

在 Chrome 或指纹浏览器里：

1. 打开扩展程序页面
2. 开启开发者模式
3. 选择“加载已解压的扩展程序”
4. 选择 `collectors\browser-extension`
5. 插件默认上报到本机中台：`http://127.0.0.1:4100`

## 重要提醒

- 不要删除 `apps/api/prisma/dev.db`，它保存中台当前消息、状态、已查看记录。
- 不要删除 `collectors/email-imap/.email-imap-state.json`，它保存邮箱采集位置，避免迁移后重复拉历史邮件。
- 不要把交付包上传到公开网盘，因为里面包含邮箱 IMAP 授权码。
- 如果移动目录后无法启动，先重新运行 `deploy\windows\install.bat`。
