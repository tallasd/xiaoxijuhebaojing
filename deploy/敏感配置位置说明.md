# 敏感配置位置说明

本交付包按要求包含真实生产配置和敏感信息。请只放在可信生产电脑上。

## 已包含的敏感文件

- `collectors/email-imap/.env`
  - 网易邮箱 IMAP 地址
  - 邮箱账号
  - IMAP 授权码
  - 每个邮箱对应的平台和店铺
  - 省流模式配置

- `apps/api/.env`
  - 数据库地址
  - API 端口
  - CORS 页面来源

- `apps/api/prisma/dev.db`
  - 当前中台数据库
  - 消息记录
  - 已查看状态
  - 店铺、平台、采集器、处理记录等数据

- `collectors/email-imap/.email-imap-state.json`
  - 每个邮箱当前采集 UID 位置
  - 用来避免迁移后重复抓旧邮件

## 安全建议

- 压缩包只在内网、U 盘、可信网盘里传输。
- 不要发给不需要维护系统的人。
- 如果邮箱授权码泄露，请到网易邮箱里重新生成授权码，并更新 `collectors/email-imap/.env`。
- 生产电脑迁移完成并确认可用后，旧电脑上的压缩包可以单独备份到加密硬盘。
