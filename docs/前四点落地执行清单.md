# 前四点落地执行清单

当前只实现前四点，远程电脑邮件中转先不作为重点。

## 1. eBay 走邮箱 IMAP

把 eBay 对应邮箱填到 `docs/邮箱清单.csv`，店铺先按：

- 扬总 eBay
- 青瓜 eBay
- yukl eBay
- Winnie 的 eBay

中台会用 IMAP 读取这些邮箱，不读取网易邮箱大师界面。网易邮箱大师继续保留给你人工查看。

## 2. Game Club 走邮箱 IMAP

把日本 Game Club / GameTrade 对应邮箱填到 `docs/邮箱清单.csv`，店铺先按：

- 老gc
- gc2
- gc3

Game Club 第一阶段先靠邮件发现新消息，后续如果网页也有明显聊天界面，再补插件兜底。

## 3. Shopee 必须装浏览器插件

Shopee 台湾 4 个店铺、Shopee 马来 1 个店铺都要在对应云端浏览器里安装插件。

插件地址保持：

```text
http://127.0.0.1:4100
```

插件里可以直接选快捷配置：

- Shopee 台湾店1
- Shopee 台湾店2
- Shopee 台湾店3
- Shopee 台湾店4
- Shopee 马来店1

以后新增 Shopee 店铺时，不需要改中台代码。直接在插件里填写新店铺信息，点击 `保存并登记到中台`，中台会自动创建这个店铺和采集器。

## 4. Discord 两个环境按插件监控

两个 Discord 聊天环境也安装同一个插件，快捷配置分别选：

- Discord 环境1
- Discord 环境2

以后新增 Discord 环境时，也直接在插件里登记。

它主要看三件事：

- 有没有客户新消息。
- 有没有超过 5 分钟未回复。
- 客服有没有敷衍或违规回复。

## 判断是否接通

打开中台 `http://localhost:5173`，看三个位置：

- `配置管理`：能看到 5 个接入路线。
- `实时监控`：能看到对应采集器。
- `质检雷达`：插件上报聊天后会出现质检观察和告警。
