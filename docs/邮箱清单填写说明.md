# 邮箱清单填写说明

这份清单用于第一阶段的邮件接入：中台电脑不读取网易邮箱大师界面，而是用每个邮箱自己的 IMAP 信息直接读取邮件。网易邮箱大师里的“所有收件箱”，对应到系统里就是“多邮箱 IMAP 同时读取”。

## 需要填写的文件

- `docs/第一阶段邮箱清单.csv`：当前优先使用，先填写 eBay 和 Game Club 的 IMAP。
- `docs/邮箱清单.csv`：通用备用表。
- `docs/SMTP发送邮箱清单.csv`：远程电脑邮件中转备用，当前前四点可以先不填。
- `docs/MAIL_ACCOUNTS_JSON模板.txt`：后面把清单转成系统配置时用。

## 先填 IMAP 收件清单

1. 打开网易邮箱大师。
2. 看左侧每一个邮箱账号，逐个抄到 `邮箱账号`。
3. 判断这个邮箱对应哪个平台，例如 eBay、Etsy、独立站等。
4. 填写这个邮箱对应的店铺名。
5. 根据邮箱服务商填写 IMAP 服务器和端口。
6. 到邮箱网页版开启 IMAP 服务，生成授权码。
7. 把授权码填到 `IMAP授权码`。
8. `是否启用` 填 `是`。
9. `是否接收未知邮件` 先填 `否`，避免网易安全提醒、广告邮件进入中台。

## SMTP 发送清单先放后面

SMTP 是给远程电脑发告警邮件用的。你现在先做本机云端浏览器和中台直连，所以这部分可以暂时不填。

1. 每台远程电脑可以先共用一个发件邮箱，部署最简单。
2. 如果后面量大、容易触发频率限制，再改成多邮箱分摊。
3. `收件邮箱（中台邮箱）` 填中台电脑会读取的邮箱。
4. `SMTP授权码` 同样不是邮箱登录密码，而是邮箱后台生成的授权码。

## 常用 SMTP 参数

| 邮箱类型 | SMTP服务器 | 端口 | SSL/TLS |
|---|---:|---:|---|
| 网易163 | smtp.163.com | 465 | SSL |
| 网易126 | smtp.126.com | 465 | SSL |
| 网易企业邮箱 | smtp.qiye.163.com | 465 | SSL |
| QQ邮箱 | smtp.qq.com | 465 | SSL |
| Gmail | smtp.gmail.com | 465 或 587 | SSL/TLS |
| Outlook/Hotmail | smtp.office365.com | 587 | TLS |

## 填写原则

- IMAP 负责“中台收平台邮件”。
- SMTP 负责“远程电脑把告警发给中台”。
- 授权码不要发给别人，也不要截图公开。
- 一个邮箱对应一个店铺最好，后续自动归类最稳。
- 一个邮箱对应多个店铺也可以先接入，但后面需要加规则拆分。
