# 插件登记新店铺流程

以后新增店铺或新增云端浏览器环境，不需要先改中台代码。

## 操作方式

1. 在新店铺所在的浏览器里安装 `collectors/browser-extension` 插件。
2. 打开插件弹窗。
3. 填写：
   - 中台地址：`http://127.0.0.1:4100`
   - 平台：例如 `Shopee TW`、`Shopee MY`、`Discord`
   - 店铺：新店铺名或聊天环境名
   - 站点：例如 `TW`、`MY`、`GLOBAL`
   - 环境名称：这台云端浏览器或运营环境的名字
4. 点击 `保存并登记到中台`。
5. 回到中台 `配置管理` 页面刷新。

登记成功后，中台会自动创建或更新：

- 平台
- 店铺/环境
- 插件采集器

之后插件每 30 秒巡检，有消息时会继续上报到这个店铺下面。

## 什么时候还要手动处理

如果一个邮箱对应多个店铺，邮件 IMAP 这边仍然需要补拆分规则。浏览器插件登记只解决网页插件这一类店铺和环境。
