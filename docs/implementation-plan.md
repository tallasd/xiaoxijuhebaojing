# 实施路线

## 第一阶段：消息雷达

目标：不漏消息、统一提醒、统一派单、统一追踪。

核心模块：

- 中央 API：消息接收、去重、风险识别、状态流转、采集器心跳。
- 客服工作台：统一收件箱、消息详情、处理动作、超时高亮。
- 邮件采集器：eBay、Etsy、日本平台邮件提醒。
- 网页采集器：Shopee、G2G 等网页未读数、红点、标题变化。
- 桌面通知采集器：千牛、微信、LINE、闲鱼等通知提醒。

## 第二阶段：客服辅助

- AI 多语言翻译、总结、回复建议。
- 知识库匹配和多语言模板。
- 自动派单、排班、移动端提醒。
- 客服绩效和风险报表。

## 第三阶段：半自动闭环

- 支持官方 API 的平台可中台直接回复。
- 与订单、发货、商品、库存系统联动。
- 自动生成售后方案、自动回访和风险复盘。
